💡 Automatic Staircase Light Project

📌 Objective
To automatically turn ON staircase lights when a person is detected and turn them OFF after some time.

🛠️ Tools Used
Arduino UNO, IR Sensors, LEDs, Relay Module, Jumper Wires.

⚙️ Working
IR sensor detects movement.
Arduino activates staircase lights.
Lights turn OFF automatically after delay.

📈 Result
Automatic lighting system with energy saving.

🚀 Applications
Homes, apartments, hotels, smart buildings.
